# Dice Roll Battle

A multiplayer Python dice game where players compete to reach 25 points.

## How It Works
- 2 to 4 players supported
- Players choose whether to roll or pass each turn
- Dice rolls between 1 and 6
- Rolling a 1 resets the playerâ€™s score to 0
- First player to reach 25 points wins

## Run
```bash
python dice_game.py
```
## Gameplay
- Enter number of players
- Each player rolls or passes on their turn
- Game ends automatically when a player wins
